## 🚀 Featured Project

### [Disaster Alert System](https://github.com/AyushiSahai/Disaster-Alert-System)
> Real-time alert system built on AWS cloud architecture

A real-time alert system built on AWS that monitors weather conditions and sends automated email alerts via SNS when dangerous conditions are detected. Integrates DynamoDB, S3, IAM and OpenWeather API.

**Stack:** `Node.js` `Express` `EC2` `DynamoDB` `SNS` `S3` `IAM`
